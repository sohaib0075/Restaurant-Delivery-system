# 🍔 Restaurant Delivery System

A console-based Restaurant Delivery System developed in C++ that allows customers to browse menus, place food orders, manage deliveries, and generate bills efficiently. This project demonstrates Object-Oriented Programming (OOP) concepts, file handling, and data structure implementation in C++.

## 📌 Features

### Customer Module
- View restaurant menu
- Search food items
- Place food orders
- Modify or cancel orders
- View order details
- Generate order bill

### Restaurant Module
- Add new food items
- Update food item details
- Remove food items
- Manage menu inventory
- View customer orders

### Delivery Module
- Assign delivery orders
- Track order status
- Update delivery progress
- Mark orders as delivered

### Additional Features
- User-friendly console interface
- File handling for data storage
- Order history management
- Input validation
- Modular code structure
